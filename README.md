##リーダブルコード勉強会

###開発言語
Ruby

###実行方法
以下のコマンドを実行してください

```
ruby app.rb
```

###レシピの追加方法

レシピのデータはyamlを使って管理しています． 
杏仁豆腐のデータを追加したい場合には

```
recipes:
  - name: オムライス
  - name: 杏仁豆腐
```

```
recipes:
  - name: オムライス
  - name: 親子丼
  - name: 杏仁豆腐
```

のように

```
- name: レシピ名
```

として保存してください

###IDの指定方法
第一引数にIDの数字を入力してください

```
ruby app.rb 2
#=>2: 親子丼
```
